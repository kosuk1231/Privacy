# 🛡️ 개인정보 및 인공지능 윤리 점검표

**서울특별시사회복지사협회**

사회복지사를 위한 개인정보 보호 및 AI 윤리 자가점검 웹 애플리케이션입니다.

## 주요 기능

### 📋 자가점검표
- 5개 영역, 24개 문항의 체계적인 점검
  - I. PC 및 정보시스템 보안 점검 (90점)
  - II. 개인정보 수집 및 보관 점검 (60점)
  - III. 정보 파기 (30점)
  - IV. 교육 및 인식 점검 (40점)
  - V. 인공지능(AI) 윤리기준 점검 (70점)
- 실시간 영역별 점수 계산 (총 290점 만점)
- 엑셀(CSV) 다운로드

### ⚙️ 관리자 대시보드
- 통계 대시보드 (제출 수, 평균/최고 점수, 영역별 분석)
- 점검 기록 관리 (필터링, 등급 표시, 삭제)
- 점검자 관리 (추가/삭제)
- 부서 관리 (추가/삭제)

## 사용 방법

### 로컬 실행
```bash
# 단순히 index.html을 브라우저에서 열면 됩니다.
open index.html
```

### GitHub Pages 배포
1. 이 저장소를 GitHub에 push합니다.
2. Settings → Pages → Source를 `main` 브랜치로 설정합니다.
3. 배포된 URL로 접속하면 바로 사용 가능합니다.

### 관리자 접속
- 우측 상단 `관리자` 버튼 클릭
- 기본 비밀번호: `1234`

## 기술 스택
- **HTML / CSS / JavaScript** (단일 파일, 프레임워크 없음)
- **Persistent Storage API** (세션 간 데이터 유지)
- 반응형 디자인 (모바일 지원)
