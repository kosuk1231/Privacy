# 🛡️ 개인정보 및 인공지능 윤리 점검표

**서울특별시사회복지사협회**

사회복지사를 위한 개인정보 보호 및 AI 윤리 자가점검 웹 애플리케이션

## 배포 방법

### Vercel 배포
1. GitHub에 이 저장소를 push
2. [vercel.com](https://vercel.com)에서 Import Project
3. 별도 설정 없이 바로 Deploy

### 관리자 접속
- 비밀번호: `1234`

## 기능
- 5개 영역 24개 문항 자가점검 (총 290점)
- 실시간 점수 계산
- 관리자 대시보드 (통계, 기록, 점검자/부서 관리)
- 엑셀(CSV) 다운로드
- localStorage 기반 데이터 저장
